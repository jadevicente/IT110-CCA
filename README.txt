INSTRUCTIONS TO RUN THE APP:
----------------------------
1. Download or clone the project files to your local machine
2. Ensure all files are in the same directory:
   - index.html (main landing page)
   - app.html (currency converter application)
   - scripts.js (JavaScript functionality)
   - styles.css (styling)
   - photo/ folder (contains team and project images)

3. Open the application:
   Method 1 - Using a web server:
   - Use a local web server (e.g., Live Server extension in VS Code, Python's http.server, or Node.js http-server)
   - Navigate to the project directory and start the server
   - Open your browser and go to http://localhost:[port]/index.html

   Method 2 - Direct file access:
   - Simply double-click on index.html to open it in your default web browser
   - Note: Some features may not work properly due to CORS restrictions when accessing files directly

4. Navigate the application:
   - index.html: Landing page with team information and project overview
   - Click "View Project" button or navigate to app.html to access the currency converter
   - app.html: Main currency converter application

5. Using the Currency Converter:
   - Enter the amount you want to convert
   - Select the source currency (From)
   - Select the target currency (To)
   - Click "Convert" button
   - View the converted amount and exchange rate information
